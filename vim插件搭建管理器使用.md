## Vundle插件管理器用法介绍：

| :PluginInstall | 安装~/.vimrc中"call vundle#begin()"到"call vundle#end()"范围内配置好的插件； |
| -------------- | ---------------------------------------- |
| :PluginClean   | 清理已经从"call vundle#begin()"到"call vundle#end()"范围删除的插件。 |
| :PluginUpdate  | 更新插件                                     |
| :PluginSearch  | 搜索插件，如":PluginSearch html"搜索包含html关键词的插件。 |
| :PluginInstall | 安装插件                                     |
